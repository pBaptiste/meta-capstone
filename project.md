- Document the progress by committing to a separate feature branch on github
- Prepare the Little Lemon web app for search engines and social media platforms, by adding meta tags and setting up Open Graph protocol (OGP).
    - Add the appropriate description (meta tag)
    - Add the og:title meta tag
    - Add the og:description meta tag
    - Add the og:image meta tag
- Commit the progress to a separate feature branch on github
- The next step is to add the navigation and routes. 
    - Update the hyperlinks in your navbar to use the Link component.
    - In the Main component you created previously, add the Routes component and define a route for each page.
- Commit the progress to a separate feature branch on github
- Style the homepage by adding the appropriate css following the brand style    guide to make it look as close to image1.png from the screenshots folder. Make sure to consider mobile responsivity as well
- Commit the progress to a separate feature branch on github
